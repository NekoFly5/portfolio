  • Socket.io
  • Three.js
  • Docker
  • JavaScript
    • 
                            

                            
                            

                                
SProjet universitaire en équipe de 3 : créer un jeu multijoueur temps réel accessible depuis mobile, où les joueurs se dissimulent parmi 25 NPCs.

                                
TPermettre à plusieurs joueurs de contrôler leur personnage via smartphone sur un écran partagé, avec 8 effets Glitch aléatoires.
